# FSD Migration Examples

Эта папка содержит готовые файлы-примеры для миграции проекта на Feature-Sliced Design архитектуру.

## Структура

```
.fsd-examples/
├── shared/              # Примеры для Shared слоя
│   ├── api/            # API базовый слой
│   ├── ui/             # UI компоненты
│   └── lib/            # Утилиты и хуки
├── entities/           # Примеры для Entities
│   ├── user/
│   └── part/
├── features/           # Примеры для Features
├── widgets/            # Примеры для Widgets
├── pages/              # Примеры для Pages
└── app/                # Примеры для App layer
```

## Как использовать

1. Скопируйте нужный файл-пример из `.fsd-examples/` в соответствующую папку в `src/`
2. Адаптируйте под свои нужды
3. Обновите импорты

## Порядок миграции

1. **ФАЗА 0**: Создать структуру папок FSD + настроить aliases
2. **ФАЗА 1**: Shared слой (API + UI + Lib)
3. **ФАЗА 2**: Entity User/Auth
4. **ФАЗА 3**: Entity Part
5. **ФАЗА 4**: Остальные Entities
6. **ФАЗА 5**: Features
7. **ФАЗА 6**: Widgets
8. **ФАЗА 7**: Pages
9. **ФАЗА 8**: App layer
10. **ФАЗА 9**: Обновление импортов
11. **ФАЗА 10**: Тестирование
